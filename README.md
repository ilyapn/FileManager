* Мной были выполнены оба дополнительных задания.
* В качестве БД я использовал MySQL, работоспособность с другими БД не гарантирую.
* Для того, что бы вам подключиться в БД измените поля url, user и password в методе db класса TreeBasicView.
* Приложение обновляет БД раз в 5 секунд, в блоке pr:poll файла index.xhtml можно изменить это значение.
* Из-за сложности реализации функционла, не предусмотреного в PrimeFaces, у меня не получилось реализовать иконку загрузки и корректную задержку развертывния узла(наверно стоило использовать GWT).
* Кликнув на узел левой конпкой мыши, он будет отмечен цветом.
* Кликнув правой кнопкой мышки по узлу можно выбрать операции delete, rename и new folder.
* Узлы можно перетаскивать зажав левую кнопку мышки.
* Дерево кешируется, то есть узел открытый единожды, при следующих открытиях не будет открываться две секунды.Что бы это изменить, в блоке pr:tree файла index.xhtml, нужно поменять параметр cache с false на true.
